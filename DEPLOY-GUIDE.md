# GitHub Pages Deployment Steps for DVK Telugu Typing Website

## Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Name: `dvk-telugu-typing`
3. Make it **Public**
4. Do NOT add README (we have files already)
5. Click **Create Repository**

## Step 2: Push Files
Run these commands in terminal:

```bash
cd "C:\Users\Good Morning\Desktop\dvk-telugu-typing"
git init
git add .
git commit -m "DVK Telugu Typing Website - Professional Telugu Typing Service"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/dvk-telugu-typing.git
git push -u origin main
```

## Step 3: Enable GitHub Pages
1. Go to your repo on GitHub
2. Click **Settings** → **Pages**
3. Source: **Deploy from branch**
4. Branch: **main** → Folder: **/ (root)**
5. Click **Save**

## Step 4: Wait & Access
- Wait 2-5 minutes
- Your site will be live at:
  **https://YOUR_USERNAME.github.io/dvk-telugu-typing**

## Step 5: SEO Submission
After site is live:
1. Submit to Google: https://search.google.com/search-console
2. Submit to Bing: https://www.bing.com/webmasters
3. Add sitemap: https://YOUR_USERNAME.github.io/dvk-telugu-typing/sitemap.xml
